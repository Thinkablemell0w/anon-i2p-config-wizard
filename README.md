# anon-i2p-config-wizard
cli config wizard for I2P


Settings list:

**Bandwidth settings**
  * Upload
  * Download
  
**Tunnel settings**
  * Length of In-Out Tunnels
  * Quantity 
  * Backups
  
**I2Psnark settings**
  * Length of In-Out Tunnels
  * Quantity 
  * autostart	
  
**Router Console settings**
  * Password
  * Graphs
  * Logs
  * Advanced Console
  * summaryBar
  * search engine (if one works again)
  
**Webapps settings**
  * i2psnark
  * susimail
  * jetty
  
**RAM settings**
  * set RAM
  
**Privoxy settings**
  * trustfile
  * tlds
  * blocks
